shell中的一些trick

shell中的一些trick

利用()或{}形成代码块

()和{}都可以对一连串的命令放在一起作为一个整体执行。括号里面的某个命令的重定向只影响该命令,但括号外的重定向则影响到括号里的所有命令。

除了一些共同点之外，两者也有一些不同点。

  1. ()最后一个命令可以不用分号，{}最后一个命令后要跟分号或者换行符
  2. ()在shell中是操作符，因此它和内部的命令之间可以不用空白符分割,{}是shell的保留字，因此它和内部的命令之间必须有空白符分开
  3. ()是对一串命令重新开一个子shell进行执行,{}对一串命令在当前shell执行
# 下面代码中，()中对var赋值操作由于是在新的shell中进行的，因此对这里的var变量无影响
var=0
(echo "in ()";var=1)
echo "var=$var"     # var=0

{ echo "in {}";var=2;}
echo "var=$var"     # var=2

$((exp)) POSIX标准的扩展计算

$((exp))这种格式中，exp可以是c语言支持的所有运算符，不过仅仅针对于整型的计算。

echo $((1+2))       # 3

a=3
b=4
echo $((a>b?a:b))   # 4

$((b++))            # 4
$((a+=3))           # 6

$((a==6))

((exp))就等价于 let "exp"
