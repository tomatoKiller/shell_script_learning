shell中的数学运算

shell中的数学运算

shell中，有多种方式可以执行数学运算

整型运算

let命令，只支持整数运算。表达式中的变量前不需要$。如果表达式中包含了空格或其他特殊字符，则必须用双引号引起来。

a=4
b=5
let result=a+b # 这里算式内部不能有空格
echo $resut # 9

let "result = a + b" # 如果有空格，需要用双引号括起来
echo $resut # 9

let a++
echo $a     # 5

let a+=5    # 这里+=前后不能有空格
echo $a     # 10

let a=a**2
echo $a     # 100

(())符号，只支持整数运算

a=4
b=5
((a++))

echo $((a+b))   # 10

((a+=5))
echo $a         # 10

((a=a**2))
echo $a         # 100

[ ]符号，只支持整数运算，[ ]内的空格可以省略，这点和if语句后的condition有区别。另外，表达式中的变量前不需要$。[ ]这种方式为过时用法，不建议使用了。

a=4
b=5
echo $[ a + b ]     # 9

echo $[ a++ ]       # 4
echo $a             # 5

echo $[ a += 5 ]    # 10
echo $a             # 10

#方括号内的变量引用可以带$，也可以省略
echo $[$a+7]        # 17

echo $[a**2]        # 289

expr命令， 只支持整数运算。注意 expr在进行乘法操作是要对*号进行转义，expr不支持幂乘运算。

a=4
b=5
result=`expr $a + $b`   #空格不能省略，$不能省略
echo $result            # 9

a=`expr $a + 1`
echo $a                 # 5

expr $a \* 10           # 50

浮点数运算

浮点数运算需要借助bc命令

# 简单运算
a=54
echo "$a * 0.3" | bc    # 16.2

# 指定精度
a=44
echo "$a / 3" | bc          # 14
echo "scale=3;$a / 3" | bc  # 14.666

进制转换

需要借助bc命令

a=22

# 十进制转二进制
echo "obase=2;$a" | bc      # 10110

# 二进制转十进制, 注意 obase和ibase的位置顺序不能调换
b=10110
echo "obase=10;ibase=2;$b" | bc  # 22
