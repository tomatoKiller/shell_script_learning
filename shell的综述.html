shell的综述

shell的综述

shell本身可以看做是宏处理器。用来对文本或者符号进行扩展来实现复杂的功能。Unix shell不仅是一个命令解释器，同时还是一种编程语言，拥有着强大的功能。作为一种命令解释器，shell中可以运行各种gnu命令，同时shell内部提供了一些实用的命令。而作为一种编程语言，shell允许把这些命令集整合在一起实现强大而复杂的功能。 shell有多种不同的类型。一般来说，Unix/Linux系统都支持多种风格的shell。 比较常见有Bourne Shell、Bash、C Shell、Z Shell、Korn Shell、TC Shell等等。本课程主要讲解Bash，后文中的shell如无特别说明，均指的是Bash。

在linux终端中，可以使用命令来改变当前的shell。

bash        # 切换当前shell为Bash
zsh         # 切换当前shell为zsh

Bash

Bash的全名是Bourne Again SHell。它是在sh(Bourne Shell)基础上发展起来的，在很大程度上兼容sh的shell，同时它从ksh和csh中引入了一些有用的特性。

另外，不同的Bash版本支持的指令集会有差别，可以通过下面的命令来查看当前Bash的版本

bash --version

shell执行流程

  1. 从 文件中/命令字符串中/terminal中 读取输入
  2. 对读取到的输入进行切分，得到token。
  3. 把得到的token转换成命令。命令可以是简单的形式(如一个命令后跟若干个参数)或复杂的形式(如管道拼接的多个命令、循环语句等等)。
  4. 执行shell扩展，把扩展后的的token拆成文件名、命令、参数等。
  5. 执行重定向，然后删除掉重定向操作符合操作数
  6. 执行命令
  7. 等待命令执行结束，获取退出状态

bash的 POSIX mode

启动bash的POSIX模式有三种方式：

  1. 启动bash时指定--posix
  2. 在bash环境中执行set -o posix
  3. 执行sh命令来进入bash（会自动进入POSIX模式)
