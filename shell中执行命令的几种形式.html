shell中执行命令的几种形式

shell中执行执行命令的几种形式

在shell中，有时需要外部传入一个命令字符串，在shell脚本内部执行。在处理这种问题时，不同的处理方式会有不同的效果。

直接执行命令

在shell中，对于确定的命令，可以直接执行。

  • 如果不需要获取命令的返回值，则可以如下所示直接在shell脚本中使用命令。
cd /home/zicky
make
./install

执行字符串中的命令

脚本中有时会需要执行一个字符串中存放的命令，对于一些简单命令，依然可以使用上面提到的方法。然而，当命令比较复杂，比如有下面情况的时候，就无法再使用上面的方法

  • 用管道组合起来的多个命令
  • 命令包含重定向符号

这个时候可以使用shell内置的eval命令。eval命令在实际执行参数中的命令之前，会先对参数进行扫描，并且对参数中的变量进行一次替换，然后再把替换后的参数用来执行。

# 变量的间接访问
val_a="aaaa"
val_b="val_a"
echo $val_b             # val_a
eval echo \$$val_b      # aaaa

# 各种命令
cmd="echo hhh > text.txt"
eval $cmd

cmd="nohup java -jar service.jar > /dev/null 2>&1 &"
eval $cmd
