函数说明

函数说明

学习函数之前，先来看看shell脚本中，命令的执行顺序：

  1. 查找别名，如果所执行命令是某个命令的别名，则执行对应的真实命令
  2. shell关键字
  3. shell script内部自定义的函数
  4. shell内置脚本
  5. shell外部命令
# type命令可以查看真正命令的来源
type pwd        # pwd is a shell builtin
type vim        # vim is /usr/bin/vim

函数定义

函数定义有两种方式：

function fun1() {
    echo "fun1"
}

fun2() {
    echo "fun2"
}

如上，函数定义时，关键字function可以省略

unset -f fun1   # 删除函数fun1的定义
declare -f      # 查看当前shell会话中定义的函数

函数的定义和引用可以分离

使用source命令或者.命令可以把其他文件中定义的函数引入到当前会话中。

#function.sh
fun1() {
    echo "I am fun1"
}

#main.sh
source function.sh  # 也可使用 . function.sh
fun1                # 输出 I am fun1

函数的参数、变量和返回值

函数中的参数

和shell脚本的位置参数一样，shell函数也使用位置参数来读取传入的参数，另外，函数内部的位置参数是函数私有的，对其操作不影响函数外的变量。

下面的函数，通过$1来应用传递给他的第一个参数

say_hello() {
    echo "hello $1"
}

say_hello "small white cat"     # hello small white cat

函数中的变量

函数中有两种变量

  • local定义的函数局部变量，只能在函数内部访问，外部同名变量不受影响
  • 函数外部定义的全局变量，函数内可以访问并且修改
outer_arg="zicky"
inner_arg="ttt"

fun1() {
    local inner_arg="sivolin"
    echo "hello $outer_arg and $inner_arg"
    outer_arg="zzzz"
}

fun1        # hello zicky and sivolin

echo "hello $outer_arg and $inner_arg" # hello zzzz and ttt

函数的返回值

函数如果不主动执行return命令，则默认返回最后一条命令执行的结果

fun1() {
    echo "aaa"
    pwd
    # 此处不使用return命令，则默认返回命令pwd的执行状态
}

函数也可以在末尾使用return命令来指定返回值，返回值只能是1-255之间的数字，负数返回值会被自动转换到这个区间，转换方式为256+返回的负数

fun1() {
    return 1
}
fun1
echo $?     # 1

fun2() {
    return -2
}
fun2
echo $?     # 254

注意： shell中，函数或者命令的执行结果状态，0表示正常结束，非0表示有错误发生。这一点跟其他的语言可能会有所区别，尤其在if条件语句中使用命令的结束状态时，要尤为注意，这点后面会讲到

在shell中执行完命令或函数时，实际上是使用waitpid内核函数获取到的命令或函数的退出状态。如果函数时被信号kill掉的话，实际获取到的退出状态是128+n（n是信号值）。

脚本的退出

在函数中如果执行exit命令，则当前脚本直接退出，exit参数值即为脚本的最终状态

fun1() {
    echo "aaa"
    exit 0          # 脚本退出，状态为0
}
